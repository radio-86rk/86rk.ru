Интерфейс

Интерфейс редактора состоит их трёх частей:

  • таблица символов -- поле, разбитое на 256 ячеек, в котором отображаются все символы шрифта, с помощью мышки или клавиатуры производится выбор редактируемого символа, а также через эту таблицу производится блочное редактирование символов: копирование, очистка, удаление, вставка;
  • матрица символа -- поле, в котором отображается попиксельное строение символа, здесь производится редактирование символа;
  • панель параметров -- панель для настроек параметров отображения, параметров шрифта, выбора загруженного шрифта, параметров сохранения.

Панель параметров имеет следующие функции:

  • Знакоместо -- задаёт размер матрицы символа для отображения, эту функцию необходимо использовать, когда программа неправильно определила размер символа и требуется ручной выбор корректного размера из списка, а также для конвертирования размера символа;
  • Таблица -- задаёт размер шрифта в 128 или 256 символов;
  • Файлы -- меню со списком открытых файлов;
  • Тип -- задаёт тип шрифта: для прямого типа пиксель зажигается единичным битом, для инверсного -- нулевым битом.

Редактор позволяет загружать несколько шрифтов и копировать блоки символов из одного шрифта в другой. Для открытия файла достаточно мышкой перетащить файл со своего компьютера в окно редактора (drag&drop).

Редактор автоматически вычисляет размер символа загруженного шрифта, размер таблицы и тип шрифта. Если редактор неправильно вычислил какой-либо параметр, его легко можно изменить, выбрав нужное значение в панеле параметров.

Управление файлами

Ctrl+O -- загрузить файл

Ctrl+S -- сохранить файл

Ctrl+Q -- закрыть файл

Ctrl+E -- создать пустой файл

Ctrl+[ или PageUp -- выбрать предыдущий файл из списка

Ctrl+] или PageDown -- выбрать следующий файл из списка

Ctrl+0..9 -- выбрать файл из списка по номеру

Ctrl+Shift+U -- переместить текущий файл вверх по списку

Ctrl+Shift+D -- переместить текущий файл вниз по списку

Ctrl+Shift+O или Alt+F3 -- открыть диалоговое окно для загрузки файла шрифта с сервера

Ctrl+Shift+S или Alt+F2 -- открыть диалоговое окно для записи множества шрифтов в один файл

Редактирование символа с клавиатуры

Стрелки -- переместить курсор

Пробел -- инвертировать точку

Shift+стрелки -- сдвинуть символ или все выделенные символы в пределах знакоместа

Ctrl+C -- очистить знакоместо или все выделенные символы

Ctrl+I -- инвертировать символ или все выделенные символы

Редактирование символа мышкой

Клик -- инвертировать точку

Перемещение курсора с зажатой левой кнопкой -- инвертировать точки

Перемещение курсора с зажатой левой кнопкой + Ctrl -- ставить точки

Перемещение курсора с зажатой левой кнопкой + Alt -- стирать точки

Перемещение курсора с зажатой правой кнопкой -- стирать точки

Дополнительные комбинации для редактирования

Ctrl+Shift+стрелка влево -- сдвинуть левую часть символа по вертикальной линии курсора влево

Ctrl+Shift+стрелка вправо -- сдвинуть правую часть символа по вертикальной линии курсора вправо

Ctrl+Shift+стрелка вверх -- сдвинуть верхнюю часть символа по горизонтальной линии курсора вверх

Ctrl+Shift+стрелка вниз -- сдвинуть нижнюю часть символа по горизонтальной линии курсора вниз

Ctrl+Alt+стрелка влево -- сдвинуть правую часть символа по вертикальной линии курсора влево

Ctrl+Alt+стрелка вправо -- сдвинуть левую часть символа по вертикальной линии курсора вправо

Ctrl+Alt+стрелка вверх -- сдвинуть нижнюю часть символа по горизонтальной линии курсора вверх

Ctrl+Alt+стрелка вниз -- сдвинуть верхнюю часть символа по горизонтальной линии курсора вниз

Выбор символа

Буквенно-цифровая клавиша -- быстрый выбор символа

Home -- переместиться в начало текущей строки символов

End -- переместиться в конец текущей строки символов

Ctrl+Home -- переместиться в начало таблицы символов

Ctrl+End -- переместиться в конец таблицы символов

Ctrl+стрелки -- переместиться влево/вправо/вверх/вниз по таблице символов

Вставка/удаление/копирование

Insert -- вставить пустой символ в текущую позицию со сдвигом всех символов вправо (в сторону увеличения кода)

Delete -- удалить символ в текущей позиции со сдвигом всех символов влево (в сторону уменьшения кода)

Shift+Insert -- работает в двух режимах: 1. если не было копирования символов: вставить пустую линию в текущую позицию со сдвигом символов на одну линию вниз (в сторону увеличения кода); 2. если было копирование символов: вставить скопированный диапазон символов в текущую позицию со сдвигом

Shift+Delete -- удалить текущую линию со сдвигом всех символов на одну линию вверх (в сторону уменьшения кода)

Ctrl+A -- выделить все символы таблицы

Shift+Click -- выделить диапазон символов от текущей позиции до новой позиции по клику

Ctrl+Insert -- копировать выделенный диапазон символов в буфер

Ctrl+Delete -- удалить скопированные символы из буфера

Shift+PageUp -- сдвинуть текущий символ на одну позицию вверх (в сторону уменьшения кода)

Shift+PageDown -- сдвинуть текущий символ на одну позицию вниз (в сторону увеличения кода)

Ctrl+Shift+PageUp -- сдвинуть текущий символ на 16 позиций вверх (в сторону уменьшения кода)

Ctrl+Shift+PageDown -- сдвинуть текущий символ на 16 позиций вниз (в сторону увеличения кода)

Ctrl+Z -- откатить изменения (undo)

Ctrl+Shift+Z -- повторить изменения (redo)
