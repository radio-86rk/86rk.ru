Всё ещё альфа, но не такая махровая, как раньше.
Умеет в Z80, недокументированные команды.
В исходнике можно прыгать по меткам, что значительно облегчает изучение программы. Кликайте на имена меток в командах переходов и вызовов подпрограмм, для возврата в исходную точку нажимайте правую кнопку мышки. Также можно просматривать список адресов, которые ссылаются на данную метку -- просто наведите мышку на метку.
Для указания областей с данными удобно мышкой выделять в дампе регионы, в меню параметров автоматически появится строка с начальным и конечным адресом выделенного региона. Добавление выделенного региона осуществляется кнопкой ">>".

Управление файлами

Ctrl+O — открыть бинарный файл

Ctrl+S — записать тектовый файл

Ctrl+Q — закрыть текущий файл

Ctrl+0..9 — выбрать загруженный файл

Ctrl+Alt+S — записать бинарный файл

Ctrl+Shift+S — записать файл проекта

Dump

Ctrl+Right — переместить курсор на следующую команду

Ctrl+Left — переместить курсор на предыдущую команду
