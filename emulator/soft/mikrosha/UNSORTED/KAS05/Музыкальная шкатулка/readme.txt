Сначала загрузить саму программу в эмулятор, затем, 
не запуская программу, загрузить по директиве "I" одно из приложений.

Запуск программы - G6000 Enter
Запуск проигрывания - Shift+I (И) Enter

Для загрузки следующей мелодии - выйти в Монитор через Shift+K (К) Enter
и загрузить по директиве "I" следующее приложение. 
Затем запустить программу снова G6000 Enter
